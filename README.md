This is a Streamlit-based Applicant Tracking System (ATS) that supports:

- Job creation with required skills
- Bulk resume upload (PDF / DOCX)
- Automatic skill matching and shortlisting
- Interview selection based on match percentage
- Bulk Excel upload
- Hiring funnel and dashboard

Built using Python, Streamlit, and SQLite.

This project is intended for demos, pilots, and small to mid-sized recruitment teams.
